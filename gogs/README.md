## gogs

Gogs 是一个可以自己托管的Git服务，类似于GitHub，GitLab等等平台。他是一个[开源](https://github.com/gogits/gogs)项目，主要使用 Go 语言。

## usage

### 直接使用 gogs 镜像

具体可参考官网教程：<https://github.com/gogits/gogs/blob/master/docker/README.md>

### 使用 docker compose

    docker-compose up -d
