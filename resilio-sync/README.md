# resilio-sync

cp env to `.env` and edit the `.env`:

```
# resilio sync config path
RESILIO_SYNC_CONFIG_PATH=path/to/config
# resilio downloads path
RESILIO_SYNC_DOWNLOADS_PATH=path/to/downloads
# resilio sync path
RESILIO_SYNC_PATH=path/to/sync
# path to password file, the content in the file will used as password
PASSWORD_FILE_PATH=./mypassword
# the domain point to your resilio admin
YOUR_DOMAIN=www.example.com
# your email to generate Let's Encrypt SSL
YOUR_EMAIL=admin@example.com
```
